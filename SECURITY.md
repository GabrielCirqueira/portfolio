# Security Policy

## Supported Versions

As this is a personal portfolio website, only the latest version deployed to the `main` branch is supported.

| Version | Supported |
| ------- | ------------------ |
| Main    | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to Gabriel Cirqueira via [gabrielcirqueira711@gmail.com](mailto:gabrielcirqueira711@gmail.com). All security vulnerabilities will be promptly addressed.

Please do NOT create a public GitHub issue for security vulnerabilities.
