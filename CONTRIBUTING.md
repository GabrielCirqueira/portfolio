# Contributing

Thanks for being interested in my portfolio! As this is a personal project, I am not actively seeking major contributions or feature requests. However, if you spot a typo, a bug, or have a suggestion for improvement, feel free to open an issue or a pull request.

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone the project** to your own machine.
3.  **Commit changes** to your own branch.
4.  **Push your work** back up to your fork.
5.  **Submit a Pull Request** so that I can review your changes.

## Development Setup

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Run the development server:
    ```bash
    npm run dev
    ```

## Code Style

This project uses `biome` for linting and formatting. Please run `npm run format` and `npm run lint` before submitting your code.
